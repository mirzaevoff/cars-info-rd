# Cars Info RD
Welcome to the Cars Info RD repository on GitHub. Here you can browse the source, look at open issues and keep track of development.

## Shortcodes
We use shortcode system to display our functions easily for users

| Shortcode | Description |
| ----- | ----- |
| `[rd_getspecs]` | This shortcode displays input form to decode VIN code, and gives car specification information. |
